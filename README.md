# Todo-app

## Use cases
A todo app project. Actor will be able to 
- add Todo-task,
- set Todo-task complete,
- remove Todo-task, 
- set Todo-task theme,
- hold Todo-tasks in local storage
- register,
- login,
- synchronize local Todo-tasks with account Todo-tasks.

## Implemented for now:
- add Todo-task,
- set Todo-task complete,
- remove Todo-task, 
- set Todo-task theme,
- hold Todo-tasks in local storage
